# LeadGateway

Outbound calling platform for US sales, appointment setting, and lead generation.

## Tech Stack

- **Frontend:** React, Vite, Tailwind CSS
- **Backend:** Node.js, Express (future)
- **Database:** MongoDB Atlas (future)
- **Voice/SMS:** Twilio (future)

## Prerequisites

- Node.js (v18 or higher)
- npm

cd frontend
npm install
